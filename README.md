# Computer Science Fundamentals

## Overview

This repository contains my notes and learning resources on core Computer Science subjects. The goal is to build a strong understanding of the concepts that form the foundation of software engineering and computer systems.

## Topics

- **Computer Organization & Architecture** – CPU, memory, instruction execution, and computer hardware fundamentals.
- **Operating Systems** – Processes, threads, scheduling, memory management, file systems, and synchronization.
- **Database Management Systems (DBMS)** – Relational databases, SQL, normalization, indexing, transactions, and concurrency.
- **Computer Networks** – OSI/TCP-IP models, HTTP, TCP/UDP, routing, DNS, and network security.
- **Object-Oriented Programming (OOP)** – Classes, objects, inheritance, polymorphism, abstraction, encapsulation, and design principles.
- **Software Engineering** – SDLC, software design, testing, version control, and development methodologies.
- **Artificial Intelligence (AI)** – Search algorithms, machine learning basics, and introductory AI concepts.
- **System Design** – Scalability, load balancing, databases, caching, messaging systems, and system architecture.

## Purpose

This repository serves as a personal knowledge base for learning, revision, and interview preparation.
